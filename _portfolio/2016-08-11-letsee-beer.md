# Letsee Service
- 주제: Letsee Beer
- 증강현실 맥주 리뷰 서비스 
- 참여 기간: 2015년 11월 ~ 2016년 8월
- 담당 직무: 
    + 맥주가 인식되기 위한 맥주 데이터 수집 가공 처리
    + 수집된 맥주 정보가 서비스에 사용될 수 있게 추가/수정/삭제가 가능한 관리 페이지 구축
    + 렛시비어 어플리케이션(iOS/Android)을 위한 맥주 정보 서비스 API
